# Spartan 6 FPGA Dev Board

Đây là thiết kế của một board FPGA dựa trên Xilinx Spartan-6 XC6SLX16, kiểu chân BGA 324 chân. Board tương thích với slot mini PCI Express. Trên board tích hợp 256MB DRAM DDR3, 1Gb NAND Flash và 32Mb NOR Flash phục vụ cho mục đích lưu trữ file cấu hình FPGA.
Board này phục vụ cho mục đích học tập, nghiên cứu liên quan tới FPGA.
